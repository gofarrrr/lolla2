# Quick Reference – Design System Usage\n\n**Use this guide when building new components. For detailed rules, see `DESIGN_SYSTEM_SPEC.md`**\n\n---\n\n## COLORS (Use Only These)\n\n### Text & Backgrounds (Neutral)\n```\ntext-ink-1       #0F172A  (headings, primary text)\ntext-ink-2       #334155  (body copy)\ntext-ink-3       #64748B  (subtle, muted)\ntext-label       #64748B  (small labels)\n\nbg-canvas        #F3F4F6  (page background)\nbg-white         #FFFFFF  (card backgrounds)\nborder-mesh      #C9CED6  (dividers, subtle borders)\n```\n\n### Accents (Frames & Interaction Only)\n```\naccent-green     #68DE7C  (primary CTA, focus, active)\naccent-orange    #FF6B3D  (urgent, high-priority, accent CTA)\naccent-yellow    #EAE45B  (medium-priority, secondary alerts)\n```\n\n---\n\n## COMPONENTS\n\n### Primary Button\n```html\n<button className=\"bg-white text-ink-1 border-2 border-accent-green rounded-2xl px-8 py-3 font-semibold hover:shadow-[0_0_0_6px_rgba(104,222,124,0.25)] hover:-translate-y-0.5\">\n  Primary Action\n</button>\n```\n\n### Secondary Button\n```html\n<button className=\"bg-white text-ink-1 border border-mesh rounded-2xl px-8 py-3 font-medium hover:shadow-md\">\n  Secondary Action\n</button>\n```\n\n### Input Field\n```html\n<input className=\"w-full px-4 py-3 border border-mesh bg-white rounded-lg text-text-body focus:outline-none focus:ring-2 focus:ring-accent-green/20 focus:border-accent-green\" />\n```\n\n### Card\n```html\n<div className=\"bg-white border border-mesh rounded-2xl p-6 shadow-sm hover:shadow-md\">\n  Card content\n</div>\n```\n\n### Badge (High Priority)\n```html\n<span className=\"bg-white border border-accent-orange text-ink-1 px-2 py-0.5 rounded text-xs font-medium\">\n  HIGH PRIORITY\n</span>\n```\n\n### Badge (Medium Priority)\n```html\n<span className=\"bg-white border border-accent-yellow text-ink-1 px-2 py-0.5 rounded text-xs font-medium\">\n  MEDIUM\n</span>\n```\n\n### Badge (Default/Info)\n```html\n<span className=\"bg-white border border-mesh text-text-label px-2 py-0.5 rounded text-xs font-medium\">\n  INFO\n</span>\n```\n\n### Active Navigation Item\n```html\n<button className=\"border-l-2 border-accent-green text-ink-1 py-2 px-3 rounded-lg\">\n  Active Item\n</button>\n```\n\n### Inactive Navigation Item\n```html\n<button className=\"border-l-2 border-transparent text-text-body py-2 px-3 rounded-lg hover:bg-cream-bg/40\">\n  Inactive Item\n</button>\n```\n\n---\n\n## BORDERS & SHADOWS\n\n| Element | Border | Radius | Shadow |\n|---------|--------|--------|--------|\n| Button (primary) | 2px green | 2xl | hover glow |\n| Button (secondary) | 1px mesh | 2xl | hover: shadow-md |\n| Input | 1px mesh | lg | none at rest |\n| Card | 1px mesh | 2xl | shadow-sm hover:shadow-md |\n| Badge | 1px [accent or mesh] | sm | none |\n| Nav active | 2px green left | lg | none |\n| Nav inactive | transparent | lg | none |\n\n---\n\n## DO's ✓\n\n✓ Use white backgrounds for cards and buttons  \n✓ Use 1px mesh border for inputs and cards  \n✓ Use 2px accent border for primary CTAs and active states  \n✓ Use accent colors only for borders/frames  \n✓ Use text-label for small, muted text  \n✓ Apply focus ring globally (auto via globals.css)  \n✓ Keep shadows minimal at rest, subtle on hover  \n✓ Use rounded-2xl for buttons and cards  \n✓ Center typography vertically with line-height  \n✓ Test color contrast (WCAG AA minimum)\n\n---\n\n## DON'Ts ❌\n\n❌ Fill buttons/cards with accent colors  \n❌ Color text with accent colors (exceptions: underlines, badges)  \n❌ Use custom colors (stick to design tokens)  \n❌ Add borders to non-interactive elements  \n❌ Use 3px+ borders  \n❌ Apply shadows to inputs at rest  \n❌ Mix multiple accent colors in one component  \n❌ Use orange or yellow for primary CTAs  \n❌ Add flashy colored frames to asides/sidebars  \n❌ Neglect focus states (all interactive elements need green ring)  \n\n---\n\n## TAILWIND ALIASES (Legacy)\n\nIf you see these in old code, update to new tokens:\n\n```\ncream-bg         → canvas\nwarm-black       → ink-1\nborder-default   → mesh\nborder-focus     → accent-green\ntext-body        → text-ink-2\ntext-heading     → ink-1\nbright-green     → accent-green\nsoft-persimmon   → accent-orange\nbrand-lime       → accent-green\n```\n\n---\n\n## FOCUS STATES (Auto)\n\nAll interactive elements automatically get:\n\n```css\nfocus:outline-none\nfocus:ring-2\nfocus:ring-accent-green\nfocus:ring-offset-2\nfocus:ring-offset-canvas\n```\n\nDefined in `app/globals.css` – **no need to add manually**.\n\n---\n\n## TYPICAL PAGE STRUCTURE\n\n```html\n<div className=\"min-h-screen bg-canvas\">\n  <!-- Header/Nav -->\n  <nav className=\"bg-white border-b border-mesh\">...</nav>\n  \n  <!-- Main content -->\n  <main className=\"py-8 px-4\">\n    <div className=\"max-w-container mx-auto space-y-6\">\n      <!-- Cards, forms, etc. -->\n    </div>\n  </main>\n  \n  <!-- Footer -->\n  <footer className=\"bg-white border-t border-mesh\">...</footer>\n</div>\n```\n\n---\n\n## COMMON PATTERNS\n\n### Form with Error\n```html\n<div>\n  <label className=\"text-xs font-semibold text-ink-1\">Email</label>\n  <input \n    className=\"... focus:ring-accent-orange focus:border-accent-orange\"\n    aria-invalid={hasError}\n  />\n  {hasError && <p className=\"text-xs text-accent-orange mt-1\">Error message</p>}\n</div>\n```\n\n### Card with Status Badge\n```html\n<div className=\"bg-white border border-mesh rounded-2xl p-6\">\n  <div className=\"flex justify-between items-start\">\n    <h3 className=\"text-lg font-semibold text-ink-1\">Title</h3>\n    <span className=\"bg-white border border-accent-green text-ink-1 px-2 py-1 rounded text-xs\">\n      Active\n    </span>\n  </div>\n  <p className=\"text-text-body mt-2\">Content</p>\n</div>\n```\n\n### Loading State\n```html\n<button disabled className=\"... disabled:opacity-50 disabled:cursor-not-allowed\">\n  {isLoading ? 'Loading...' : 'Submit'}\n</button>\n```\n\n---\n\n## NEED HELP?\n\n1. **Color question?** → Check COLORS section above\n2. **Component specs?** → See COMPONENTS section\n3. **Border/shadow rules?** → See BORDERS & SHADOWS table\n4. **Detailed rules?** → Read `DESIGN_SYSTEM_SPEC.md`\n5. **Audit checklist?** → Use Section 6 of DESIGN_SYSTEM_SPEC.md\n\n---\n\n**Last Updated:** Oct 15, 2025  \n**Version:** 1.0\n
